# Image Strategy — Cash4HomeFL

## Overview

Every page on Cash4HomeFL should have 3–5 images appropriate to its content type.
This document defines the image strategy: what image each page type needs, whether it
should be AI-generated or free stock, and the recommended prompt for generation.

**Rule of thumb:** Use real photos where trust and authenticity matter (homepage hero,
seller stories, reviews). Use illustrations/infographics where explanation and contrast
matter (how it works, comparison, process). Use maps/diagrams where location matters.

---

## Image Types

| Type | Style | Use When |
|------|-------|----------|
| Editorial real estate photo | Warm, natural South Florida light | Hero sections, seller stories, reviews |
| Before/after illustration | Side-by-side split graphic | Situation pages (foreclosure, damaged) |
| Process infographic | Minimal line-art steps | How it works section |
| Comparison illustration | Two-panel contrast | Cash vs. listing comparison |
| Map graphic | Clean flat vector, brand colors | County/city service area |
| Pull-quote author photo | Circular crop, neutral background | Testimonial section |
| Stats illustration | Bold number + icon | Outcome stats (days to close, etc.) |

## AI Image Generation Prompts — Master List

### Hero Photos
```
"A beautiful single-family home in [CITY/PALM BEACH COUNTY, FLORIDA], golden hour warm light,
professional real estate photography, lush palm trees in the yard, clean well-maintained exterior,
magazine-quality, ultra-realistic, no text, no people"
```

### Before/After Splits
```
"Split screen photograph: left half shows an older Florida home with peeling paint, overgrown
yard, visible deferred maintenance — right half shows the same home fully renovated with fresh
paint, clean landscaping, new roof. Clean white divider line in center. Photo-realistic."
```

### Process Infographics
```
"Clean minimalist infographic on white background showing [3 or 4] steps:
[step descriptions]. Each step has a simple line-art icon (house, document, handshake).
Brand color #1D4ED8 used for icons and step numbers. Text in clean sans-serif.
No photographs, vector illustration style only."
```

### Comparison Graphics
```
"Two-panel flat vector illustration on white background.
Left panel labeled 'Traditional Sale': house with 'For Sale' sign, long timeline arrow
labeled '60–90+ days', icons showing repair bills, agent commission (crossed out), staging costs.
Right panel labeled 'Cash Sale': house with handshake icon, short timeline arrow labeled '7–30 days',
single checkmark for 'No repairs, No fees, No commission'.
Clean, professional, brand blue #1D4ED8 accent. No photographs."
```

### Pull Quote Author Photos
```
"Professional circular headshot on neutral background, diverse person in their [30s/40s/50s],
friendly confident expression, wearing casual business casual, natural lighting.
Crop to circle. No text. Ultra-realistic photograph."
```

### Bold Stats Graphics
```
"Bold typographic poster: large number '[STAT]' in weight 800 sans-serif, brand blue #1D4ED8,
with a small simple house icon. Below: '[STAT LABEL]' in regular weight.
Clean white background, minimal, print-quality, no photographs"
```

### Maps
```
"Clean flat vector map of [Palm Beach County and Broward County / specific city] Florida.
Cities marked with small circles. Water shown in light blue. Brand blue #1D4ED8 used
for highlights. White background. Minimalist style, no street names cluttering the map."
```

## Implementation

Images are generated using `/scripts/generate-site-images.py` and saved to `/public/images/`.

Output directory structure:
```
public/images/
├── homepage/
│   ├── hero.jpg
│   ├── how-it-works-infographic.jpg
│   ├── comparison.jpg
│   └── stats-graphic.jpg
├── city/
│   └── [city-slug]/
│       ├── hero.jpg
│       ├── aerial.jpg
│       └── stats.jpg
├── county/
│   └── [county-slug]/
│       ├── aerial.jpg
│       └── map.jpg
├── situation/
│   └── [situation-slug]/
│       ├── hero.jpg
│       └── before-after.jpg
├── blog/
│   └── [slug]/
│       └── featured.jpg
└── shared/
    ├── process-infographic.jpg
    ├── comparison.jpg
    └── map-pbc-broward.jpg
```

Run generation with:
```bash
python scripts/generate-site-images.py city west-palm-beach
python scripts/generate-site-images.py county palm-beach-county
python scripts/generate-site-images.py situation foreclosure
python scripts/generate-site-images.py homepage
```

See `/scripts/generate-site-images.py` for full batch generation and API key handling.
